## 제이쿼리 기본 문법
* `$()` 제이쿼리 기본 선언법
* `$('.클래스명')` 
* `$('#아이디명')` 
* `$('태그명')` 
* `''` 따옴표 안 CSS 선택자처럼 원하는 대상 작성 가능
* `$('대상').show();` 대상 보이기
* `$('대상').hide();` 대상 숨기기
* `$('대상').css('속성','값');` CSS 적용하기
* `$('대상').removeClass();` class 제거하기 (괄호 비우면 모든 클래스 제거)
* `$('대상').addClass('추가클래스명');` class 적용하기
